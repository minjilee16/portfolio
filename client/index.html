| Splitter

Optical character recognition web and mobile application for splitting bills

Splitter is the best app that allows you to split your bill among friends. It uses optical character recognition and a regex-based algorithm to parse receipt items and price from images. This app allows users to assign items to different people (users and non-users), which then receive a text message with the total amount they need to pay, along with a pre-filled Venmo link to pay the ‘splitter’.

TECH STACK: React, Bootstrap, Redux, React Router, Twilio, Postgres, Passport, Knex, Bookshelf, Redis, Heroku, Google Vision
